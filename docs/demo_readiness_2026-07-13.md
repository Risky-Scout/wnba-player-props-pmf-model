## Known Limitations

1. **Market edge is not CLV**: `model_edge_over` = model probability minus current no-vig market probability. True CLV requires archived closing quotes, which are not yet fully operational.

2. **Market snapshots not archived**: Historical CLV cannot be computed. Only `model_edge_vs_open_agreement_rate` is available from `backtest_clv.py` (correctly labeled as non-CLV).

3. **Combo props independence assumption**: Points + rebounds, etc. are generated by convolving independent marginals. Joint simulation is architecturally specified but not yet implemented. All combo props are labeled EXPERIMENTAL.

4. **Injury data is current-state**: The model uses current injury status for live predictions. Historical injury snapshots are not archived; historical rows use only lagged injury-count features.

5. **No wager execution log**: All CLV metrics are hypothetical (no actual wager history).

6. **Calibrator staleness**: If calibrators are > 90 days old, they are labeled STALE and predictions are downgraded to WATCHLIST.
